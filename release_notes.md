# 🚀 Uviewer Release Notes

## 🇰🇷 한국어 (Korean)
<ko-KR>
### **Uviewer - 올인원 미디어 뷰어**
문서, 만화, 음악, 비디오를 하나의 앱에서 간편하게 즐기세요!

**✨ 주요 기능:**
*   **📚 강력한 문서 뷰어**: epub, txt, pdf 및 아오조라 문고(aozora) 형식 지원. 세로 쓰기, 루비 문자, 강조점(Bouten) 완벽 구현.
*   **🔍 스마트 인코딩 감지**: 한글(EUC-KR, 조합형), 일본어(Shift-JIS) 등 텍스트 인코딩 자동 감지로 글자 깨짐 방지.
*   **🖼️ 압축 파일 즉시 보기**: zip, cbz, rar, 7z 내 이미지를 압축 해제 없이 실시간으로 감상.
*   **📖 다양한 보기 모드**: 두 페이지 보기(LTR/RTL), 분할 보기, 확대 배율 고정(Zoom Lock) 기능 제공.
*   **🌐 WebDAV 스트리밍**: 개인 서버(NAS, Nextcloud 등)의 파일을 다운로드 없이 실시간 스트리밍으로 감상.
*   **🎨 프리미엄 디자인**: Material 3 기반의 세련된 UI, 부드러운 애니메이션, 완벽한 다크 모드 지원.
</ko-KR>

## 🇺🇸 English
<en-US>
### **Uviewer - All-in-One Media Viewer**
Enjoy documents, manga, music, and videos all in a single app!

**✨ Key Features:**
*   **📚 Powerful Document Viewer**: Supports epub, txt, pdf, and Aozora Bunko formats. Full support for vertical writing, Ruby text, and emphasis dots.
*   **🔍 Smart Encoding Detection**: Automatically detects UTF-8, EUC-KR, and Shift-JIS to prevent text corruption.
*   **🖼️ Instant Archive Viewing**: View images inside zip, cbz, rar, and 7z files without extraction.
*   **📖 Flexible View Modes**: Dual page (LTR/RTL), Split view, and Zoom Lock functionality.
*   **🌐 WebDAV Streaming**: Stream files directly from your private cloud (NAS, Nextcloud) without downloading.
*   **🎨 Premium Design**: Sleek Material 3 UI with fluid animations and full Dark Mode support.
</en-US>

## 🇯🇵 日本語 (Japanese)
<ja-JP>
### **Uviewer - オールインワンメディアビューア**
ドキュメント、漫画、音楽、動画を一つのアプリで快適に！

**✨ 主な機能:**
*   **📚 強力なドキュメントビューア**: epub、txt、pdf、青空文庫形式に完全対応。縦書き、ルビ、傍点を美しく再現。
*   **🔍 スマート文字コード判定**: Shift-JIS、EUC-KR、UTF-8などを自動判別し、文字化けを徹底防止。
*   **🖼️ アーカイブ即時閲覧**: zip、cbz、rar、7z内の画像を解凍することなく、そのままスムーズに閲覧可能。
*   **📖 多彩な表示モード**: 見開き表示（左開き/右開き）、左右分割表示、拡大率固定（Zoom Lock）機能を搭載。
*   **🌐 WebDAVストリーミング**: 個人サーバー（NAS、Nextcloudなど）からファイルをダウンロードせずにリアルタイムでストリーミング再生。
*   **🎨 プレミアムデザイン**: Material 3ベースの洗練されたUI、滑らかなアニメーション、ダークモードに完全対応。
</ja-JP>
